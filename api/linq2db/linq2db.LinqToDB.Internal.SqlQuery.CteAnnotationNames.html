Remarks

Expected value type is bool. Annotation values participate in both query-cache keying (via GetElementHashCode() and the IExpressionCacheKey-marked CteAnnotationsContainer) and LinqService round-trips, so values must hash stably and serialize via the standard SerializationConverter. Non-primitive values are not supported today.

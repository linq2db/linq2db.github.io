Properties

CompiledWrappers

Provides access to delegates, created from expressions, defined in wrapper class using following property and type mappings, configured for TypeMapper:

private static IEumerable<T> Wrappers { get; }

where T could be LambdaExpression or Tuple<LambdaExpression, bool>. Boolean flag means that mapping expression compilation allowed to fail if it is set to true. This could be used to map optional API, that present only in specific versions of provider. If wrapper doesn't need any wrapper delegates, this property could be ommited.

    RetrieveIdentity<T>(IEnumerable<T>, DataConnection, Boolean)

    Initializes source columns, marked with IsIdentity with values. If column had sequence name set using SequenceNameAttribute and useSequenceName set to true, values from sequence used. Otherwise column initialized with values, incremented by 1 starting with max value from database for this column plus 1.
